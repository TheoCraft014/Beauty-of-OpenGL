# Beauty-of-OpenGL
Using C++ and OpenGL API I made a noctural forest scene full of fireflies and a watchtower.
It has a textured terrain, trees animated with a wind effect, fireflies that emit point 
lights, and a watchtower equipped with spotlight lights that project a texture pattern (light cookie).
It also has fog effects.

# API install requirements:
 The following API's are needed to be installed before using the framework:
- assimp
- freetype
- GL
- GLFW
- glm
-  pfd
- spdlog
- stb
- ft2build.h

# Using GFXFramework
You can also put the "Body" directory inside the GFXFramework.
